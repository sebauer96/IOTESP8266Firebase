## Requirements Hardware

ESP8266 NODEMCU

Lamp

Arduino IDE
Install Plugin ESP8266 for Arduino IDE
Install ESP8266WiFi Library on Arduino IDE
Install FirebaseESP8266 Library on Arduino IDE
